# Agent Library Reference for Architect Agent

This document compiles agent descriptions from multiple libraries to serve as a comprehensive reference for the Architect Agent. It contains the capabilities, specialties, and recommended use-cases for various sub-agents. 

## How to use this document
When building multi-agent orchestrations, review these categories to select the most suitable agents for tasks requested by the user. Pair agents from different divisions to achieve full-stack end-to-end delivery (e.g. `Backend Architect` + `Frontend Developer` + `QA Expert` + `DevOps Automator`).
